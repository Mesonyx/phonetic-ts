# phonetic-ts

Generates random, pronounceable (but gibberish) words.

A TypeScript update for [@TomFrost](https://github.com/TomFrost)'s [phonetic.js](https://github.com/TomFrost/node-phonetic) package. 

It has been altered to work with TS projects and no longer relies on the (now deprecated) [crypto](https://www.npmjs.com/package/crypto) library.

## Installation

`npm install phonetic-ts`

## Usage

> import { phonetic } from 'phonetic-ts';

See [original package](https://github.com/TomFrost/node-phonetic) for usage instructions.
